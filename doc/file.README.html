nhentai-api

nhentai-api is a basic and easy to use API for nhentai.net

Installation

Gem install

gem install nhentai-api

Gemfile

gem 'nhentai-api'

API

doujinshi = Doujinshi.new(220794)
puts doujinshi.title
puts doujinshi.pages

puts Tag.listing('ahegao')
puts Character.listing('2b')

Documentation

You can read the documentation here: https://www.rubydoc.info/github/Mraiih/nhentai-api
